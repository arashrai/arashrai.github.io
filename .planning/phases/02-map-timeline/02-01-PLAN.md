## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | MAPBOX_TOKEN in public JS | accept | Token is inherently public on a static site with no server. Restrict to arashrai.com domain in Mapbox dashboard. Free tier limits exposure. |
| T-02-02 | Tampering | Mapbox GL JS CDN integrity | accept | Pin to specific version (v3.9.4). Mapbox CDN is widely used and well-maintained. SRI hash could be added but may break on CDN updates. |
| T-02-03 | Denial of Service | Rapid scroll flooding map.flyTo | mitigate | map.stop() called before each flyTo to cancel in-progress animations. Stop index change gating prevents per-pixel scroll triggers. |
| T-02-04 | Spoofing | Auth bypass to Our Story page | mitigate | NARSH_AUTH.requireAuth() runs before map init. Redirects to gate if no tier in localStorage. Same pattern as all section pages. |
</threat_model>
